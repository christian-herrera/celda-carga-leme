App para Celda de Carga

Conexión Serial

Esta aplicación permite conectarse por comunicación serial únicamente al modelo especifico del modulo para la cual se diseñó. Una vez establecida la conexión se puede realizar alguna de las siguientes operaciones:

  • Se permite cofigurar el valor de Constante de Calibración. Este valor se usa para establecer el factor con el cual se ajusta la curva de la balanza.
  • Se permite cofigurar el valor de Peso Conocido.
  • Se puede descargar los valores anteriores usando los botones correspondientes.
  • Se permite iniciar el proceso de Calibración Automática. Este proceso permite calcular en el Arduino, la constate correspondiente para un Peso Conocido
  • Se permite iniciar la recolección de los valores de forma continua. A medida que llegan se cargan en el gráfico y se muestra en los demas campos.

Al momento de exportar, se tienen 4 variables:

  • param.peso_conocido: Valor en KG del peso usado para la constante de calibración.
  • param.cte_calib: Constante asociada a la variable anterior.
  • data_time: Vector tiempo, expresado en segundos.
  • data_kg: Vector de muestras de peso, expresado en Kg
